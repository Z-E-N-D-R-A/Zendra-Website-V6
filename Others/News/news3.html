Zentral-Chat Emergency Maintenance (v6.0.1)
15 February 2026
With the launch of Zentral-Chat few days ago, we have discovered serveral security loopholes. We have already patched it with this update (v6.0.1), and would like to talk about the issue and our solution in this article.
Security Updates
In order to allow account recovery on Zentral-Chat v6.0.0, we have assigned each user with a recovery code and accountId. With that code, you will be able to unlock the accountId and login into your account. This is a simple yet effective recovery system. However, it creates a few privacy and security issues.

First of all, accountId is not directly tied to your browser. This would allow anonymous users to access your account and data since there is no way of restricting them. In addition, to allow recovery code verifications within frontend, all recovery codes had to be made public. Anyone can find someone else's recovery code and login into their accounts.

Our goal is to create a free and safe chat service for our community. However, with these two main issues, accounts are insecure and we know recovery code system is not ideal. Therefore, we temporarily disabled Zentral-Chat few days ago to rework the account system completely.

Accounts are now linked with browser by default. Only you in your browser has access to your account and see your data. And to make it possible to recovery accounts or login on other devices, we have added the option to connect your account with Google. In that case, your account will be linked to your Google account instead, allow you to access your account on multiple devices by logging into Google. If you would like to connect Google, you may still sign in as anonymous just like before, except it is impossible to recovery account once lost.

But, since accounts system has been completely reworked, all previously created accounts and sent messages have been removed. Please create a new account if you would like to continue chatting. Sorry for the inconveniences.

With this update, we have resolved the security issues and allow a safer place for you to chat.
Other Changes
Apart from account and security updates, we have also improved a few features in Zentral-Chat.
  • Added Terms & Condition and FAQ buttons in sidebar
  • Added sidebar animations when expanded/collapsed
  • Added jump to latest button when scrolled up
  • Added keyboard shortcut for opening Settings (Control/Command + ,)
  • Added Google account profile icon as option when connected
  • Improved new message indicator with better visual & display of no. of messages
  • Improved visual of reaction badges
  • Improved visual of settings
  • Fixed presence system showing unknown users
  • Fixed indicator bar acting weirdly
  • Fixed username not being case sensetive
  • Fixed input bar highlight when selected
  • Fixed incorrect report message limit
